# HR-Data-Analysis-
**HR Analytics Dashboard using Power BI with interactive visualizations, DAX measures, and data modeling for data-driven HR decisions**.

**>>>** 📌 **Project Overview**
This project presents an interactive HR Analytics Dashboard built in Power BI to analyze employee data and workforce trends. The dashboard provides actionable insights into employee attrition, job satisfaction, demographics, salary distribution, and overall HR performance.

**>>>** 📊**Key Features**
- Employee Attrition Analysis
- Department-wise Employee Distribution
- Gender Diversity Analysis
- Age Group Analysis
- Education Field Analysis
- Job Role Performance
- Salary Distribution
- Employee Count KPIs
- Interactive Filters & Slicers
- Business Insights for HR Decision-Making

**>>>**🛠 **Tools Used**
- Power BI
- Power Query
- DAX
- Data Modeling
- Excel

**>>>** 📈 **Dashboard Insights**
- Analyze employee attrition trends.
- Identify departments with the highest turnover.
- Compare employee distribution by gender and age.
- Monitor salary and job satisfaction metrics.
- Support HR managers with data-driven decisions.
